# Triage Labels

Matt Pocock skills 使用五个规范角色描述 Issue 的 triage 状态。
本仓库直接使用默认字符串:

| Skill role | GitHub label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整定义,Agent 可独立领取 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 使用语义名称(例如“AFK-ready 标签”)时,必须使用本表右侧
对应的 GitHub label,不要创建同义标签。
