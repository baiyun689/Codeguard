# Codeguard

Codeguard 的领域语言，描述代码审查执行过程及其可观测结果。

## Language

**审查执行流**:
一次审查中，LangGraph 节点、ReAct 推理步骤和工具调用按实际因果关系组成的完整路径。
_Avoid_: 事件列表、调用日志

**执行步骤**:
审查执行流中可独立检查输入、输出与状态变化的一个阶段，包括图节点、模型决策、工具调用和工具结果。
_Avoid_: 卡片、日志行

**主执行流**:
由顶层 LangGraph 节点组成的审查路径，用于概览一次审查的阶段、顺序与结果。
_Avoid_: 完整事件流

**审查员泳道**:
ReviewCouncil 内属于同一审查员的 ReAct 执行步骤序列，与其他审查员的并行过程分栏展示。
_Avoid_: 审查员日志

**状态变化**:
一个执行步骤通过节点输出 patch 对审查状态字段发起的写入；并行写入在 superstep 边界统一合并。
_Avoid_: 节点输出

**执行实例**:
某个图节点、模型或工具在一次审查中的单次实际运行；同一节点的重复运行对应不同执行实例。
_Avoid_: 节点、事件类型

**节点输入**:
一个 LangGraph 节点开始执行前收到的完整审查状态快照。
_Avoid_: 变更字段

**节点输出**:
一个 LangGraph 节点本次直接返回、等待合并进审查状态的字段 patch。
_Avoid_: 节点结束状态、完整状态

**Superstep 结束状态**:
同一 LangGraph superstep 内所有并行节点的输出 patch 统一合并后形成的完整审查状态。
_Avoid_: 节点结束状态、节点输出

**模型决策**:
一次 LLM 请求及其可观测响应，包括消息、工具声明、工具选择、结构化内容与用量信息；不包含模型未暴露的隐藏思维链。
_Avoid_: 模型思考、完整思维链

**状态演进**:
一次审查中，同一审查状态字段跨执行实例发生的连续变化记录。
_Avoid_: 状态快照、事件时间线
